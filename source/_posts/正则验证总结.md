# 验证大于0的整数(整数不包括0)
<!-- more -->
``` javascript
/^[1-9]\d*$/.test(value)
```
# 验证邮箱地址

``` javascript
/[\w!#$%&'*+/=?^_`{|}~-]+(?:\.[\w!#$%&'*+/=?^_`{|}~-]+)*@(?:[\w](?:[\w-]*[\w])?\.)+[\w](?:[\w-]*[\w])?/.test(value)
```

# 验证18位身份证号

``` javascript
/^(\d{6})(\d{4})(\d{2})(\d{2})(\d{3})([0-9]|X)$/.test(value)
```
# 验证11位以1开头的手机号

``` javascript
/^1\d{10}$/.test(value)
```

# 验证银行卡号16位或19位号码

``` javascript
/^(\d{16}|\d{19})$/.test(value);
```
